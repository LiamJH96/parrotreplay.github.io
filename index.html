Capture, tweak and replay live audio snippets on the fly

Parrot Replay is a low-latency audio utility aimed at streamers and gamers. Record short clips, apply pitch and speed transforms and replay them with hotkeys.

Setup Guide

What it does

  • Record short live clips from your microphone or virtual audio cable
  • Transform pitch, speed and reverse audio in real-time
  • Hotkeys map clips to the numpad for instant recall
  • Profiles & Favourites save different setups and restore on launch

Quick setup

  1. Install VB-Audio Virtual Cable (Parrot uses virtual cables for passthrough).
  2. Install the app and run once to allow drivers and permissions.
  3. Open Parrot Replay, press Rescan Devices then Start Engine.
  4. Map your hotkeys and test microphone passthrough using Test Microphone.
  5. Visit the Preferences to enable startup/minimise to tray options.

If you run into audio device issues, check Windows sound settings and ensure VB-Audio cables are "Ready" and not disabled.
